# Motivation
Twitter is one of the great information gathering tool.  
You guys wanna use twitter during your working time.

Worc is made for you.  
A quiet twitter client made by Electron.

# How To Develop
## Setup
```
$ npm install
```

## Build
```
$ npm run build
```

## Watch and auto build
```
$ npm run watch
```

## Test
```
$ npm test
```
